# Glosario

| Español | English | Notas |
| --- | --- | --- |
| Hilo | Thread | Tema de discusión creado por un usuario. |
| Publicación | Post | Mensaje individual dentro de un hilo. |
| Respuesta | Reply | Respuesta a otra publicación. |
| Categoría | Category | Agrupa hilos por temática, admite jerarquía. |
| Adjunto | Attachment | Archivo subido en una publicación. |
| Usuario | User | Rol base del foro. |
| Moderador | Moderator | Rol con permisos de moderación. |
| Administrador | Admin | Máximo rol con acceso total. |
| Rol | Role | Enum: user, moderator, admin. |
| Iniciar sesión | Sign In | Flujo de autenticación. |
| Registrarse | Sign Up | Proceso de creación de cuenta. |
| Conteo de vistas | View Count | Cantidad de veces que se vio un hilo. |
| Conteo de respuestas | Reply Count | Total de respuestas en un hilo. |
| Hilo fijado | Pinned Thread | Hilo destacado en la parte superior. |
| Paginación | Pagination | Usa la estructura `links` y `meta` del API. |
| Actualización optimista | Optimistic Update | UI se actualiza antes de confirmar en el servidor. |
| Markdown | Markdown | Formato de texto enriquecido soportado en posts. |
| ForumApp | ForumApp | Nombre del proyecto de ejemplo. |

Consulta este glosario para mantener traducciones consistentes.
