# Glossary

| English | Español | Notes |
| --- | --- | --- |
| Thread | Hilo | Discussion topic created by a user. |
| Post | Publicación | Individual message within a thread. |
| Reply | Respuesta | Reply to a post; may show quoted content. |
| Category | Categoría | Groups threads by theme; supports hierarchy. |
| Attachment | Adjunto | File uploaded with a post. |
| Moderator | Moderador | Elevated role with moderation permissions. |
| Admin | Administrador | Highest role with full management access. |
| User | Usuario | Standard forum participant. |
| Role | Rol | Enum: user, moderator, admin. |
| Sign In | Iniciar sesión | Authentication entry point. |
| Sign Up | Registrarse | Account creation flow. |
| View Count | Conteo de vistas | Number of times a thread was seen. |
| Reply Count | Conteo de respuestas | Total replies within a thread. |
| Pinned Thread | Hilo fijado | Highlighted thread shown on top. |
| Pagination | Paginación | Uses links/meta structure from API. |
| Optimistic Update | Actualización optimista | UI updates before server confirmation. |
| Markdown | Markdown | Rich text formatting supported in posts. |
| ForumApp | ForumApp | Sample application name. |

Use this table as reference for consistent translations across both languages.
