# UI Mockups & Design

Before writing any code, let's study **what we're building visually**. These mockups will guide every component we create.

## Design System

### Color Palette (Dark Theme)

```css
/* Background colors */
--forum-bg: #0f172a;           /* slate-900 - Main background */
--forum-surface: #1e293b;      /* slate-800 - Cards, modals */
--forum-hover: #334155;        /* slate-700 - Hover states */

/* Text colors */
--forum-text: #e2e8f0;         /* slate-200 - Primary text */
--forum-text-muted: #94a3b8;   /* slate-400 - Secondary text */

/* Border & dividers */
--forum-border: #334155;       /* slate-700 */

/* Brand colors */
--forum-primary: #38bdf8;      /* sky-400 - Links, buttons */
--forum-primary-hover: #0ea5e9;/* sky-500 - Hover state */

/* Semantic colors */
--forum-success: #10b981;      /* emerald-500 - Success states */
--forum-danger: #ef4444;       /* red-500 - Errors, delete */
--forum-warning: #f59e0b;      /* amber-500 - Warnings */
```

### Typography

- **Headings**: Inter font, bold (font-weight: 700)
- **Body**: Inter font, regular (font-weight: 400)
- **Code**: Fira Code, monospace

### Spacing

Using Tailwind's 4px-based scale:
- `xs`: 4px (p-1)
- `sm`: 8px (p-2)
- `md`: 16px (p-4)
- `lg`: 24px (p-6)
- `xl`: 32px (p-8)

## Interactive States

### Hover Effects
- Cards: Background changes to `#1e293b`
- Buttons: Background darkens slightly
- Links: Underline appears

### Focus States
- Inputs: Blue ring (`ring-2 ring-forum-primary`)
- Buttons: Outline appears

### Loading States
- Skeleton screens (pulse animation)
- Spinner for buttons
- Progress bar for forms

## Responsive Design

### Breakpoints
- Mobile: < 768px (1 column)
- Tablet: 768px - 1024px (2 columns)
- Desktop: > 1024px (sidebar + main content)

### Mobile Adjustments
- Hamburger menu for navigation
- Stack filters vertically
- Simplified thread cards
- Bottom navigation bar

## Accessibility

- Semantic HTML (`<nav>`, `<main>`, `<article>`)
- ARIA labels for icon buttons
- Keyboard navigation support
- Focus management for modals
- Color contrast ratio > 4.5:1
