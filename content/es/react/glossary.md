# Glosario

| English | Español | Notas |
| --- | --- | --- |
| Thread | Hilo | Discussion topic creado por un usuario. |
| Post | Publicación | Mensaje individual dentro de un thread. |
| Reply | Respuesta | Respuesta a un post; puede mostrar contenido citado. |
| Category | Categoría | Agrupa threads por tema; soporta jerarquía. |
| Attachment | Adjunto | Archivo subido junto a un post. |
| Moderator | Moderador | Rol elevado con permisos de moderación. |
| Admin | Administrador | Rol más alto con acceso completo de gestión. |
| User | Usuario | Participante estándar del foro. |
| Role | Rol | Enum: user, moderator, admin. |
| Sign In | Iniciar sesión | Punto de entrada de autenticación. |
| Sign Up | Registrarse | Flujo de creación de cuenta. |
| View Count | Conteo de vistas | Número de veces que se ha visto un thread. |
| Reply Count | Conteo de respuestas | Número total de respuestas dentro de un thread. |
| Pinned Thread | Hilo fijado | Thread destacado que se muestra arriba. |
| Pagination | Paginación | Usa estructura de links/meta de la API. |
| Optimistic Update | Actualización optimista | La UI se actualiza antes de la confirmación del servidor. |
| Markdown | Markdown | Formato de texto enriquecido soportado en posts. |
| ForumApp | ForumApp | Nombre de la aplicación de ejemplo. |

Usa esta tabla como referencia para mantener traducciones consistentes entre ambos idiomas.
